# /btn

套用或創建按鈕風格。支援「選單」、「名字/數字」、「關鍵字」、「網址仿製」四種使用方式。

## 用法範例

```
/btn                        → 顯示選單，引導選擇
/btn --help                 → 顯示選單，引導選擇
/btn flash                  → 直接輸出「閃光」風格
/btn 5                      → 直接輸出第 5 套
/btn 我要箭頭效果            → 關鍵字配對推薦
/btn https://example.com    → 分析網址按鈕並仿製
/btn flash 顏色改成 #FF5500  → 指定風格 + 自訂顏色
```

---

## Step 1：判斷輸入類型

### A. 無參數或帶 --help / -h → 顯示選單

若使用者輸入 `/btn`, `/btn --help`, `/btn -h`，輸出以下選單，等設計師回覆名字或數字：

```
📦 請選擇按鈕風格（輸入名稱、數字，或描述效果）：

| 編號 | 名稱 | 效果說明 |
| :---: | :--- | :--- |
| **1** | `skew` | 右下角斜切 — 缺角遮罩，俐落現代 |
| **2** | `arrow` | 普通箭頭 — FontAwesome 箭頭 + Hover 彈跳 |
| **3** | `thin-arrow` | 細箭頭 — CSS 折線箭頭 + 拉伸動畫 |
| **4** | `dot` | 圓點放大 — 小圓點 Hover 膨脹填滿背景 |
| **5** | `flash` | 閃光效果 — 光影由左至右快速掃過 |
| **6** | `skew-fill` | 傾斜填滿 — Hover 背景斜向填滿 |

💡 也可以：
  - 描述效果   → /btn 我要有光的效果
  - 貼網址     → /btn https://example.com
  - 自訂顏色   → /btn flash 顏色改成 #FF5500
```

### B. 帶名字或數字 → 直接跳 Step 3

名字對應表（以下皆可觸發）：
| 數字 | 名稱 | 同義關鍵字 |
|------|------|-----------|
| 1 | `skew` | 斜切、缺角、遮罩 |
| 2 | `arrow` | 箭頭、普通箭頭、fa、fontawesome |
| 3 | `thin-arrow` | 細箭頭、折線、細線 |
| 4 | `dot` | 圓點、球、展開、膨脹 |
| 5 | `flash` | 閃光、光影、掃光、閃 |
| 6 | `skew-fill` | 斜向填滿、傾斜、fill |

### C. 帶網址 → 分析仿製

1. 讀取目標網址，找出 `.btn` 相關的 CSS 樣式
2. 分析 hover 效果（動畫、顏色過渡、形狀變化）
3. 按照系統模板格式重製（含 o_cc 變數、系統按鈕排除）
4. 詢問：「要加入按鈕庫存成第 7 套並命名嗎？」

---

## Step 2：確認顏色（選填）

- **預設**：使用 `o_cc1`~`o_cc5` 主題色變數（推薦，跟隨主題切換）
- **自訂**：設計師指定特定顏色碼，AI 直接寫入（不跟隨主題）
  - 提醒設計師：「此按鈕顏色將固定，主題切換時不會變更」

---

## Step 3：讀取知識庫並輸出

讀取 `.agent/skills/icb_page_generator/resources/button_styles.md`，
取得對應套組的完整 SCSS。

**輸出格式：**
```scss
//------------------ 按鈕，[風格名稱] 開始 ------------------//

[完整 SCSS，含 o_cc1~o_cc5、排除系統按鈕、編輯模式禁用動畫、尺寸適配]

//------------------ 按鈕，[風格名稱] 結束 ------------------//
```

---

## Step 4（選填）：新增到按鈕庫

若設計師要求保存新風格：
1. 詢問命名（英文 slug + 中文名）
2. 更新 `button_styles.md` 加入新的套組 Section
3. 更新本檔案（`btn.md`）的選單和名字對應表

# btn

套用或創建 Odoo 按鈕風格。

## 用法
- `btn` 或 `btn --help` → 顯示選單
- `btn flash` 或 `btn 5` → 直接輸出
- `btn 我要箭頭` → 關鍵字配對
- `btn https://...` → 分析仿製
- `btn flash 顏色改 #FF5500` → 指定風格 + 自訂色

## 6 套命名庫
| # | 名稱 | 中文 | 效果 |
|---|------|------|------|
| 1 | `skew` | 斜切 | 右下角缺角遮罩 |
| 2 | `arrow` | 普通箭頭 | FA 箭頭 + 彈跳 |
| 3 | `thin-arrow` | 細箭頭 | CSS 折線箭頭 |
| 4 | `dot` | 圓點放大 | 圓點膨脹填滿 |
| 5 | `flash` | 閃光 | 光影掃過 |
| 6 | `skew-fill` | 傾斜填滿 | 背景斜向展開 |

## Steps
1. 判斷輸入：選單 / 名字+數字 / 關鍵字 / 網址
2. 確認顏色偏好（預設 o_cc 變數；可改固定色）
3. 讀 `button_styles.md` 輸出完整 SCSS（含開頭結尾標記）
4. 若是新風格，詢問是否命名並存入庫
